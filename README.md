# 📊 Supply Chain Performance Dashboard

This project is an interactive Power BI dashboard to analyze supply chain performance.

## 🔹 Key Metrics
- Avg Logistics Delay
- Avg Inventory
- Total Transactions

## 📈 Analysis
- Shipment status (Delivered, Delayed, In Transit)
- Traffic impact on delivery delay
- Delay analysis by traffic type

## 🎨 Design
- Dark theme dashboard
- Clean and interactive visuals

## 🛠 Tools Used
- Power BI
- Data Visualization
